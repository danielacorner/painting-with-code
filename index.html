Generate a website using an AI Large Language Model like ChatGPT.

As of Nov 2023, my favourites are ChatGPT4($), Bing, Phind, Perplexity, HuggingFace

Here’s an example prompt:

generate code for a website named “Painting with Code” which includes:

  1. a navbar with a logo and links to Sessions, About, and Contact pages
  2. a full-width landing page image with a large title in front of it saying “Make websites using English & Copy+Paste”
  3. A footer with a Google Maps embed indicating the address 10 Maple Ave Smiths Falls ON, and a link to the signup form at bit.ly/painting-with-code please research and find the embed code for this address and include the complete code in your response

Here’s the website we get: bit.ly/pwc-example

As you can see it’s pretty bare-bones, but by using more advanced prompts, or by chatting and iterating with the AI, you could create a handy website for free.
